1201 Triennial Rulemaking Commenter Leaderboard

Michael Weinberg

This list is one of many companions to the Engeleberg Center report “Unbalanced Interests”. You can find all of the report extras on the report page. Using data from every 1201 Triennial from 2000-2024, the list calculates the top individual commenters. The top ten commenters are listed on this page. The full list can be found here.

"Comments" and the "Comments by Year" totals may differ because a single comment can be filed on behalf of multiple organizations. In those situations, each organization is individually listed in the "Comments by Year" section. You can find the full dataset in this repo.
